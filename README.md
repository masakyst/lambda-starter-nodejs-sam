My AWS Lambda Starter kit/NodeJS AWS-SAM version
=================================================

Get Daikufile 

    $ wget https://raw.githubusercontent.com/masakyst/lambda-starter-nodejs-sam/master/Daikufile

Setup

    $ daiku create:nodejs

Hello 

    $ babel --version
    $ sam --version

Start local server

    $ npm start

Deploy

    $ npm run app:create
    $ npm run app:package
    $ npm run app:deploy
